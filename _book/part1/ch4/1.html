Chapter 4: 快排及随机化算法

1. 快速排序

  • 由C.A.R. Hoare于1962年提出.
  • 分治算法（Divide-and-conquer algorithm）.
  • “in place”排序 (像插入（insertion）排序，但不像归并（merge）排序).
  • 非常实用(与调谐tuning).

2. 分治（Divide and conquer）

  快速排序一个"n-元素"的数组:

  1. Divide: 将数组围绕一个主x(pivot x)划分为两个子数组，使下子数组中的元素≤x≤上子数组中的元素 。

  2. Conquer: 递归地对两个子数组排序。
  3. Combine: 一些简单工作。

        Key:线性时间分区子程序

3. 分区子程序（Partitioning subroutine）

  1. 分区子程序伪代码

    PARTITION(A,p,q)  ⊳ A[p..q]
 x ← A[p]      ⊳ pivot = A[p] 
 i←p
 for j ← p + 1 to q
     do if A[j] ≤ x
         then i←i+1
             exchange A[i] ↔ A[j]
 exchange A[p] ↔ A[i]
 return i

    不变式（Invariant）:

  2. 分区过程示例

  A=[6,10,13,5,8,3,2,11],p=1,q=8 ,x=A[1]=6,i=1.

  • step1:i=1,j=2
  • step2: A[j]=10>x, go(j+1)
  • step3: j=3,A[j]=13 > x; go(j+1)</br>       j=4,A[j]=5 ≤ x; do(i=i+1=2),A[2] ↔ A[4]
  • step4: i=2,j=4; go(j+1)
  • step5: j=5,A[j]=8 > x; go(j+1)
  • step6: j=6,A[j]=3 ≤ x; do(i=i+1=3),A[3] ↔ A[6]
  • step7: i=3,j=6; go(j+1)
  • step8: j=7,A[j]=2 ≤ x; do(i=i+1=4),A[4] ↔ A[7]
  • step9: i=4,j=7; go(j+1)
  • step10: j=8,A[j] > x; go(j+1)
  • step11: j+1>q,for循环end. p=1,i=4,A[p] ↔ A[i].return i.

4. 快排算法伪代码

QUICKSORT(A, p, r)
    if p < r
        then q ← PARTITION(A, p, r) 
            QUICKSORT(A, p, q–1) 
            QUICKSORT(A, q+1, r)

初始调用:QUICKSORT(A, 1, n)

11. 随机快排

IDEA:围绕一个随机元素进行划分。

  • 运行时间与输入无关秩序。
  • 不需要对输入分布做任何假设。
  • 没有特定的输入引出最坏情况的行为。
  • 最坏的情况仅由随机数生成器的输出决定。

14. 实践中的快速排序（Quicksort in practice）

  • Quicksort是一个伟大的通用排序算法。
  • 快速排序通常比归并排序快两倍以上。
  • 快速排序可以从代码优化中获益良多。
  • 即使使用缓存和虚拟内存，快速排序也能很好地运行。
